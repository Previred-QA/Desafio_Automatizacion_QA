5. Para ejecutar el proyecto

## Se debe ubicar la clase java "CarroCompraRunner" haciendo click derecho sobre ella seleccionando la opcion "Run", esta clase se ubica ingresando en la siguiente ruta:

>DesafioPrevired_Nathaly
  >src
    >test
      >java
        >run
	    CarroCompraRunner

**** NOTA IMPORTANTE ****

Para ejecutar nuevamente el proyecto se debe modificar la direccion de correo en el archivo .json "flujoCarroCompras", este archivo se ubica en la siguiente ruta:

>DesafioPrevired_Nathaly
  >src
    >test
      >resources
        >dataJSON
	    flujoCarroCompras.json


6. Reporteria

## Los reportes se encontraran en la carpeta "reportes" luego de la ejecucion del proyecto

## El log de la ejecucion se genera en formato json y se ubica en la siguiente ruta:

>DesafioPrevired_Nathaly
  >reportes
    >Json-Report_Carro_Compra


## El reporte HTML contiene los screenshots de cada paso y se ubica en la siguiente ruta:

>DesafioPrevired_Nathaly
  >reportes
    >TestOutput+fechahora
      >Html-Report

## Los screenshots se ubican en la siguiente ruta:

>DesafioPrevired_Nathaly
  >reportes
    >TestOutput+fechahora
